# My portfolio

#### How to build dev version:

1. Go to project directory
2. Copy ".env.example.json", rename to ".env" and update
3. Run "npm install"
4. Run "npm run dev"

Main file location: "dev/index.html"

#### How to build dist version:
1. Go to project directory
2. Run "npm install"
3. Run "grunt build"

Main file location: "build/index.html"
